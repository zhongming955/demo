# Electron 是什么？
electron 是一个使用web前端技术来构建桌面应用的框架，它嵌入了Chromium 和 Nodejs，也就是允许你像在浏览器中一样访问项目，也可以让你通过node访问本地数据库，本地文件，实现更强大的功能
# 什么情况下我们可以使用 Electron
1.需要将web页面打包为桌面应用时

2.web端无法满足需求，需要计算机更底层的权限时
